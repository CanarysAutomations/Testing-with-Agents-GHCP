# T-Rex Runner Game (Playwright Compatible)

## Tech Stack
- UI: HTML, CSS, Vanilla JavaScript
- API: Node.js + Express
- Testing/Workshop Compatibility: Playwright

## Prerequisites
- Node.js (>=18)
- npm

## Setup & Run

### 1. Install backend dependencies
```bash
cd api
npm install
node server.js
```
API runs on http://localhost:3000

### 2. Run UI
Open `ui/index.html` in a browser

### 3. (Optional) Playwright
You can point Playwright to `ui/index.html` or serve it via any static server.
